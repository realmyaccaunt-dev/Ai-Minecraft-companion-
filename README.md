# 🤖 AI Companion - Minecraft Forge Mod 1.12.2

**AI Bot Companion на OpenAI для Minecraft 1.12.2!** 🚀

Встреть своего AI помощника, который управляет полноценным телом в игре. Помощник видит мир, принимает решения благодаря GPT-4 и выполняет действия в реальном времени!

## 🎯 Основные возможности

✅ **Полноценное AI тело** - AI компаньон это настоящий игрок с собственным телом и действиями  
✅ **OpenAI GPT-4 интеграция** - умная логика через API  
✅ **Полный контроль движения** - вперед, назад, прыжки, спринт, крадкой ход  
✅ **Управление взглядом** - поворот головы во все стороны  
✅ **Действия** - добыча блоков, атака мобов, взаимодействие, чат  
✅ **Восприятие мира** - видит ресурсы, мобов, игроков, освещение, время суток  
✅ **Командование** - давай команды через консоль  
✅ **RP функции** - может говорить в чат и общаться  

## 📋 Системные требования

- **Minecraft:** 1.12.2
- **Forge:** 14.23.5.2860
- **Java:** 8+
- **Gradle:** любая версия (автоматически)
- **OpenAI API ключ:** [получить здесь](https://platform.openai.com/api/keys)

## 🚀 Быстрая установка

### 1️⃣ Установи Forge 1.12.2

```bash
# Скачай отсюда:
https://files.minecraftforge.net/maven/net/minecraftforge/forge/index.html
# Выбери: 1.12.2-14.23.5.2860 (Installer)
# Запусти и выбери "Install Client"
```

### 2️⃣ Собери мод

```bash
# Linux/Mac:
./gradlew build

# Windows:
gradlew.bat build
```

### 3️⃣ Установи мод

```bash
# Скопируй JAR файл:
build/libs/AICompanion-1.0.0.jar

# В папку:
~/.minecraft/mods/
```

### 4️⃣ Запусти Minecraft и используй команды!

## 🎮 Как использовать

### Основные команды:

```bash
/companion spawn              # Спавни компаньона рядом с тобой
/companion follow             # Компаньон следует за тобой
/companion ai sk-xxxxx        # Активируй AI (вставь свой API ключ)
/companion command <text>     # Отдай команду компаньону
/companion status             # Проверь статус
/companion stop               # Останови компаньона
```

### Примеры:

#### Стартуем!

```bash
# Спавним компаньона
/companion spawn

# Активируем AI (замени на свой ключ!)
/companion ai sk-proj-...your-key-here...

# Проверяем статус
/companion status
```

#### Даём задания

```bash
# Команда компаньону
/companion command Go find wood and bring it back

# AI сам разберётся что делать!
```

#### Командуем на ходу

```bash
/companion command Watch out for mobs!
/companion command Build a shelter
/companion command Explore this area
```

## 🧠 Как это работает?

1. **Наблюдение** 👀  
   AI сканирует окружающий мир каждую секунду:
   - Видит все блоки рядом (дерево, камень, руды)
   - Видит мобов (враги, животные)
   - Видит игроков
   - Знает уровень освещения и время суток
   - Понимает свои координаты и здоровье

2. **Анализ** 🧠  
   OpenAI GPT-4 анализирует ситуацию:
   - Читает состояние мира
   - Получает команду игрока
   - Принимает решение что делать

3. **Действие** ⚡  
   AI выполняет действие:
   - Движется в нужную сторону
   - Смотрит куда надо
   - Добывает блоки
   - Атакует мобов
   - Говорит в чат

4. **Цикл повторяется** 🔄  
   Каждую секунду AI снова смотрит на мир и решает что делать!

## 🔑 Где взять OpenAI API ключ?

1. Перейди на [platform.openai.com](https://platform.openai.com)
2. Зарегистрируйся или войди в аккаунт
3. Нажми на твой профиль → "API keys"
4. Нажми "Create new secret key"
5. Скопируй ключ (выглядит как `sk-proj-...`)

**⚠️ ВАЖНО:** Не публикуй свой ключ нигде! Это как пароль!

## 💰 Стоимость использования

OpenAI API платный:
- Ты платишь за количество использованных токенов
- Для Minecraft компаньона это примерно **несколько центов в час**
- При регистрации ты получишь **$5 бесплатного кредита**
- Мониторь использование на [platform.openai.com/account/usage](https://platform.openai.com/account/usage)

## 🐛 Если что-то не работает

1. **Проверь версию Forge**
   ```
   Должна быть: 1.12.2-14.23.5.2860
   ```

2. **Проверь что мод установлен**
   ```
   Папка: ~/.minecraft/mods/

   ```
   Файл: ~/.minecraft/logs/latest.log

4. **Проверь API ключ**

   - Ключ правильно скопирован?

   - Опиши проблему
